# FocusTrackr

FocusTrackr es una aplicación desarrollada como parte de los requisitos de un proyecto académico. Su objetivo principal es ayudar a los estudiantes y profesionales a gestionar mejor su tiempo al rastrear y analizar el tiempo dedicado a diferentes tareas y actividades diarias.

## Descripción del Proyecto
Esta aplicación permite a los usuarios establecer metas de productividad, realizar un seguimiento del tiempo invertido y obtener estadísticas detalladas para ayudar a optimizar sus hábitos de trabajo. Con FocusTrackr, los usuarios pueden tomar decisiones informadas sobre cómo dividir su tiempo de manera más efectiva.

### Características Clave
- **Registro de actividades**: Los usuarios pueden registrar el tiempo dedicado a tareas específicas.
- **Generación de reportes**: La aplicación ofrece reportes gráficos que detallan cómo se distribuyó el tiempo.
- **Gestión de metas**: Se pueden establecer metas diarias o semanales y realizar un seguimiento del progreso hacia ellas.
- **Interfaz intuitiva**: Un diseño simple y fácil de usar para maximizar la experiencia del usuario.

## Tecnologías Utilizadas
El proyecto está desarrollado en **Java**, aprovechando su robustez y flexibilidad para aplicaciones multiplataforma. Además, se utiliza un enfoque modular para garantizar que el código sea escalable y fácil de mantener.
